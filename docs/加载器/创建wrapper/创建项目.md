## 使用aiges 快速创建python wrapper 项目

通过aiges创建一个名为test的项目

```shell
root@wyq:/home/wyq# python3 -m aiges create -n test
```

### 项目结构

```
root@wyq:/home/wyq# tree test/
test/
├── Dockerfile
├── README.md
├── requirements.txt
└── wrapper
    ├── test_data
    │   └── test.png
    └── wrapper.py
```
